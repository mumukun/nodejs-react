## dependencies

node 6.3.x or later and mongodb

## developing

run mongodd on a separated terminal instance:

```
mongod
```
install 
```
 npm install  || yarn install 
```
run the app:

```bash
npm run dev
```

the app api runs on `localhost:8080`
